## Goal

Get a Next.js app running locally and confirm you can edit a page and see the change.

## Preconditions

- Node.js installed\n+- A code editor\n+\n+## Steps\n+\n+1. Create a Next.js app.\n+\n+```bash\n+npx create-next-app@latest my-app --ts --app\n+```\n+\n+2. Install dependencies.\n+\n+```bash\n+cd my-app\n+npm i\n+```\n+\n+3. Start the dev server.\n+\n+```bash\n+npm run dev\n+```\n+\n+4. Edit `src/app/page.tsx` and save.\n+\n+## Success criteria\n+\n+- The dev server starts without errors\n+- Editing a page updates the UI in the browser\n+\n+## Common mistakes\n+\n+- Running commands in the wrong directory\n+- Forgetting `npm i`\n+
